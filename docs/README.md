# Project Documentation

## Purpose

This folder stores supporting documentation that is broader than one code folder.

## What Is Here

- Architecture notes, product notes, or design references when they outgrow root docs.
- Material useful for project review, presentation, and maintenance.

## When To Edit This

- Add documents here when they explain cross-cutting decisions or project context.
- Keep folder-level READMEs close to code; use this folder for bigger narratives.

## Related Folders

- `README.md` gives the project overview.
- `lib/` contains the Flutter app implementation.
