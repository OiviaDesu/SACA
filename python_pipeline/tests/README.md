# Python Pipeline Tests

## Purpose

This folder contains tests for Python research and data pipeline code.

## What Is Here

- Unit or smoke tests for ingestion, analysis, training, and export helpers.
- Checks that keep pipeline scripts reproducible.

## When To Edit This

- Add tests when changing Python pipeline behavior.
- Keep Flutter app tests under `test/`.

## Related Folders

- `python_pipeline/data/samples/` provides small fixtures.
- `python_pipeline/requirements/` lists Python dependencies.
