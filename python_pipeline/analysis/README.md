# Pipeline Analysis

## Purpose

This folder contains scripts for inspecting datasets, model outputs, and evaluation quality.

## What Is Here

- Analysis helpers for research review.
- Scripts that summarize data or model behavior before app integration.

## When To Edit This

- Edit this folder when adding new research checks or evaluation summaries.
- Keep production Flutter inference code in `lib/infrastructure/analysis/`.

## Related Folders

- `python_pipeline/training/` creates model candidates.
- `python_pipeline/export/` prepares app-ready artifacts.
