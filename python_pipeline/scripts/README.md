# Pipeline Scripts

## Purpose

This folder groups script entrypoints that support the Python research pipeline.

## What Is Here

- Utility scripts and legacy wrappers that do not belong to one active pipeline stage.

## When To Edit This

- Add scripts here when they support research operations but are not app runtime tools.
- Prefer clear names and README notes when preserving legacy behavior.

## Related Folders

- `python_pipeline/training/` contains active training scripts.
- `tools/` contains repository or app demo tools.
