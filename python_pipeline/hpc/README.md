# HPC Workflows

## Purpose

This folder contains helper scripts for running pipeline work on high-performance computing systems.

## What Is Here

- Job scripts and environment helpers.
- Legacy wrappers for older HPC workflows.

## When To Edit This

- Edit this folder when remote training or evaluation jobs change.
- Keep local app runtime code out of HPC scripts.

## Related Folders

- `python_pipeline/training/` contains training logic.
- `python_pipeline/hpc/legacy_wrappers/` preserves older wrappers.
