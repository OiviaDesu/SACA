# Model Training

## Purpose

This folder contains scripts for training SACA model candidates.

## What Is Here

- Training workflows for diagnosis classifiers or speech-related experiments.
- Scripts that consume processed datasets and produce candidate artifacts.

## When To Edit This

- Edit this folder when model training approach, features, or evaluation setup changes.
- Promote only reviewed exports into `assets/models/`.

## Related Folders

- `python_pipeline/data/processed/` supplies training data.
- `python_pipeline/export/` packages trained outputs for the app.
