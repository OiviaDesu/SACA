# Flutter Web Host

## Purpose

This folder contains static files that host the Flutter web demo for SACA.

## What Is Here

- Web manifest, favicon, icons, and bootstrap files.
- Browser-facing metadata used when the Flutter app runs on the web.

## When To Edit This

- Edit this folder when web app branding, manifest metadata, or browser shell behavior changes.
- Keep app UI and diagnosis behavior in `lib/`.

## Related Folders

- `tools/` contains the local web demo server.
- `lib/infrastructure/web/` contains web backend adapters.
