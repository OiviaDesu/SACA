# Web Icons

## Purpose

This folder contains icons used by browsers and installable web app surfaces.

## What Is Here

- PNG icon sizes referenced by the web manifest.
- Web-specific copies of the SACA app icon.

## When To Edit This

- Update this folder after changing source branding artwork.
- Keep icon filenames aligned with `web/manifest.json`.

## Related Folders

- `assets/branding/` stores source icon artwork.
- Platform icon folders store Android, iOS, macOS, and Windows equivalents.
