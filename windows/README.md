# Windows App Shell

## Purpose

This folder contains the Windows host project for the SACA Flutter app.

## What Is Here

- CMake and native runner files.
- Flutter-generated Windows build integration.
- App resources such as icons and Windows metadata.

## When To Edit This

- Edit this folder for Windows app identity, native window setup, or build configuration.
- Keep shared Flutter behavior in `lib/`.

## Related Folders

- `lib/infrastructure/window/` configures desktop window behavior.
- `windows/runner/` contains the native Windows runner.
