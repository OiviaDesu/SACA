# Windows Runner

## Purpose

This folder contains the native Windows application wrapper for SACA.

## What Is Here

- C++ runner code that starts Flutter.
- Windows resource files and app icon metadata.
- Window creation and native app lifecycle glue.

## When To Edit This

- Edit this folder for Windows-native app identity, window behavior, or resources.
- Keep Flutter UI and assessment logic in `lib/`.

## Related Folders

- `windows/flutter/` connects plugins and Flutter build logic.
- `windows/runner/resources/` stores Windows native resources.
