# Windows Runner Resources

## Purpose

This folder stores native resources used by the Windows runner.

## What Is Here

- Windows icon and resource files compiled into the app shell.

## When To Edit This

- Update this folder when Windows app icons or native resources change.
- Regenerate resources from source branding when possible.

## Related Folders

- `assets/branding/` stores source branding assets.
- `windows/runner/` references these native resources.
