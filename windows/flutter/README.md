# Windows Flutter Build Glue

## Purpose

This folder connects the Windows runner to Flutter tooling.

## What Is Here

- Generated plugin registrant files.
- CMake helper files used by Flutter Windows builds.

## When To Edit This

- Usually let Flutter tooling maintain this folder.
- Avoid manual edits unless changing Windows Flutter integration deliberately.

## Related Folders

- `windows/runner/` contains native Windows app code.
- `pubspec.yaml` controls Flutter plugins.
