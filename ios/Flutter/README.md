# iOS Flutter Build Glue

## Purpose

This folder connects the iOS Xcode project to the Flutter build system.

## What Is Here

- Generated or environment-specific Flutter build settings.
- iOS integration files used during Flutter builds.

## When To Edit This

- Usually avoid manual edits unless Flutter tooling or iOS build setup requires it.
- Prefer changing Flutter configuration through supported Flutter commands.

## Related Folders

- `ios/Runner/` contains the iOS app shell.
- `pubspec.yaml` controls shared Flutter dependencies.
