# iOS Runner

## Purpose

This folder contains the native iOS app wrapper that launches SACA.

## What Is Here

- iOS app delegate and entrypoint files.
- Asset catalogs for icons and launch images.
- Storyboards and iOS metadata files.

## When To Edit This

- Edit this folder for iOS-native app identity, launch screen, permissions, or platform behavior.
- Do not implement shared clinical flow here.

## Related Folders

- `lib/main.dart` starts the Flutter app.
- `ios/RunnerTests/` contains iOS host tests.
