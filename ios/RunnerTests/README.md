# iOS Runner Tests

## Purpose

This folder contains iOS-native tests for the host runner.

## What Is Here

- XCTest files for validating the native iOS shell.
- Tests separate from cross-platform Flutter widget and unit tests.

## When To Edit This

- Add tests here when iOS-native behavior needs coverage.
- Keep shared SACA logic tests under `test/`.

## Related Folders

- `test/` contains Flutter tests.
- `ios/Runner/` contains the iOS host code under test.
