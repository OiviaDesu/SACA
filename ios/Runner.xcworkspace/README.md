# iOS Xcode Workspace

## Purpose

This folder stores the Xcode workspace used to open and build the iOS app.

## What Is Here

- Workspace metadata that links the runner project and dependencies.
- Shared workspace data used by Xcode.

## When To Edit This

- Usually let Xcode, CocoaPods, or Flutter tooling update this folder.
- Manual edits are rarely needed for SACA app behavior.

## Related Folders

- `ios/Runner.xcodeproj/` stores project settings.
- `ios/Flutter/` connects Flutter to iOS builds.
