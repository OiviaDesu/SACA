# iOS App Shell

## Purpose

This folder contains the iOS host project for the SACA Flutter app.

## What Is Here

- Xcode project and workspace files.
- iOS runner code, assets, launch screen, and tests.
- Flutter-generated integration files used by iOS builds.

## When To Edit This

- Edit this folder for iOS permissions, app metadata, icons, launch screen, or native setup.
- Keep shared app logic in `lib/`.

## Related Folders

- `lib/` contains the cross-platform Flutter app.
- `assets/` contains bundled images, data, and models.
