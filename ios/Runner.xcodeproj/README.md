# iOS Xcode Project

## Purpose

This folder stores Xcode project metadata for building the iOS app.

## What Is Here

- Project configuration used by Xcode and Flutter.
- Shared schemes for build and test commands.

## When To Edit This

- Edit this folder through Xcode or Flutter tooling when iOS build settings change.
- Avoid manual changes unless you understand the Xcode project format.

## Related Folders

- `ios/Runner/` contains the app target files.
- `ios/Runner.xcworkspace/` is the workspace opened by Xcode.
