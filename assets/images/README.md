# UI Images

## Purpose

This folder contains images shown directly in the Flutter user interface.

## What Is Here

- Symptom illustrations.
- Body-map images.
- General UI images such as the "other symptom" visual.

## When To Edit This

- Edit this folder when visible app imagery changes.
- Keep filenames stable when widgets already reference them.

## Related Folders

- `lib/presentation/screens/` chooses when images appear.
- `assets/branding/` stores brand source artwork.
