# Branding Assets

## Purpose

This folder stores source branding files used to generate app icons and visual identity.

## What Is Here

- SACA icon artwork and brand source images.
- Files that platform icon sets can be regenerated from.

## When To Edit This

- Update this folder when the project logo or brand source artwork changes.
- Regenerate platform-specific icons after changing source artwork.

## Related Folders

- `android/app/src/main/res/` contains Android icon resources.
- `ios/Runner/Assets.xcassets/` and `macos/Runner/Assets.xcassets/` contain Apple icon sets.
- `web/icons/` contains web app icons.
