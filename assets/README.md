# App Assets

## Purpose

This folder contains files bundled into the SACA app at runtime.

## What Is Here

- Branding and icon source assets.
- Symptom images and body-map images used by the UI.
- Lexicon data and ML model bundles loaded by Flutter.

## When To Edit This

- Edit this folder when app-visible images, local data, or bundled model files change.
- Keep large generated outputs out of Git unless they are intentional app assets.

## Related Folders

- `lib/infrastructure/analysis/` loads model assets.
- `lib/infrastructure/localization/` loads lexicon assets.
