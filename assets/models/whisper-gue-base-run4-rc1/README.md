# Gurindji Whisper Candidate

## Purpose

This folder stores a Gurindji-focused Whisper model candidate for native speech-to-text work.

## What Is Here

- Whisper model assets staged for app speech recognition.
- Files used by native runtime selection and validation.

## When To Edit This

- Edit this folder when a new Gurindji speech model candidate is staged.
- Re-test voice input after any model asset change.

## Related Folders

- `lib/infrastructure/speech/` contains speech services.
- `python_pipeline/` contains data and training support.
