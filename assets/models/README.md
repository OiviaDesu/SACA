# Model Assets

## Purpose

This folder contains ML and speech model bundles shipped with or staged for SACA.

## What Is Here

- Diagnosis classifier bundles for on-device/web inference.
- Whisper or Sherpa speech model assets for native speech-to-text experiments.
- README files describing model readiness and intended use.

## When To Edit This

- Edit this folder when a model bundle is added, replaced, or promoted to runtime use.
- Keep model README notes honest about whether a bundle is active, staged, or experimental.

## Related Folders

- `lib/infrastructure/analysis/` runs diagnosis model assets.
- `lib/infrastructure/speech/` selects speech model assets.
- `python_pipeline/` trains and exports model artifacts.
