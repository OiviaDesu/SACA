# Sherpa ONNX Gurindji Whisper Candidate

## Purpose

This folder stores a Gurindji-focused Sherpa ONNX Whisper candidate model.

## What Is Here

- Speech model files intended for Gurindji transcription evaluation.
- Candidate assets that need review before clinical or community-facing use.

## When To Edit This

- Update this folder only when replacing or evaluating the Gurindji speech candidate.
- Keep notes conservative because speech quality affects user trust.

## Related Folders

- `python_pipeline/training/` supports model training workflows.
- `lib/infrastructure/speech/` wires speech input into the app.
