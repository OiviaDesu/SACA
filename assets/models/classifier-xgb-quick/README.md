# Local XGBoost Quick Bundle

This folder is reserved for the **local-only** Dart export of the winning `quick xgboost` diagnosis classifier.

Expected generated files:

- `bundle.json` - preprocessing metadata + tree dump bundle for parity/debugging
- optional export reports copied here manually if needed

Generate the bundle locally with `python_pipeline/export_xgb_to_dart.py`.

Do **not** commit generated bundle contents from this folder.
