# macOS Runner Configs

## Purpose

This folder contains macOS build configuration files.

## What Is Here

- Xcode configuration values for debug, release, and shared app settings.

## When To Edit This

- Edit this folder when macOS build settings or app metadata change.
- Keep cross-platform Dart configuration in `pubspec.yaml` or `lib/`.

## Related Folders

- `macos/Runner/` contains the macOS target.
- `macos/Runner.xcodeproj/` references these configs.
