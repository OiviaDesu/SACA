# macOS App Shell

## Purpose

This folder contains the macOS host project for the SACA Flutter app.

## What Is Here

- Xcode project and workspace files.
- macOS runner code, assets, configuration, and tests.
- Flutter integration files used by macOS builds.

## When To Edit This

- Edit this folder for macOS app metadata, icons, permissions, window setup, or native integration.
- Keep shared app behavior in `lib/`.

## Related Folders

- `lib/infrastructure/window/` configures desktop window behavior.
- `assets/` contains bundled runtime assets.
