# macOS Flutter Build Glue

## Purpose

This folder connects the macOS Xcode project to Flutter tooling.

## What Is Here

- Flutter build settings and generated integration files.
- Native build glue used by Flutter commands.

## When To Edit This

- Usually let Flutter tooling maintain this folder.
- Edit only when macOS build integration needs deliberate changes.

## Related Folders

- `macos/Runner/` contains macOS host code.
- `pubspec.yaml` controls shared Flutter dependencies.
