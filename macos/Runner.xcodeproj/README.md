# macOS Xcode Project

## Purpose

This folder stores Xcode project metadata for macOS builds.

## What Is Here

- Project settings and shared schemes.
- Build metadata used by Xcode and Flutter.

## When To Edit This

- Edit this folder through Xcode or Flutter tooling when macOS build settings change.
- Avoid manual edits unless necessary.

## Related Folders

- `macos/Runner/` contains the app target.
- `macos/Runner.xcworkspace/` is the workspace opened by Xcode.
