# macOS Xcode Workspace

## Purpose

This folder stores the Xcode workspace for the macOS app.

## What Is Here

- Workspace metadata that links the runner and dependencies.
- Shared workspace data used by Xcode.

## When To Edit This

- Usually let Xcode or Flutter tooling update this folder.
- Manual edits are rarely needed for app behavior.

## Related Folders

- `macos/Runner.xcodeproj/` stores project settings.
- `macos/Flutter/` connects Flutter to macOS builds.
