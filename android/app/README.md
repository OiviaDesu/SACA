# Android App Module

## Purpose

This folder is the main Android application module generated and maintained for Flutter.

## What Is Here

- Android build configuration for the app package.
- Source sets for debug, profile, and main builds.
- Resources used by the Android launcher and system UI.

## When To Edit This

- Change this folder when Android app metadata, signing, permissions, or resources need updates.
- Keep Flutter UI and clinical flow changes in `lib/`.

## Related Folders

- `android/gradle/` provides Gradle wrapper support.
- `windows/`, `ios/`, and `macos/` provide equivalent host shells.
