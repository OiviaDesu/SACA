# Android Resources

## Purpose

This folder contains Android resource files used by the native shell.

## What Is Here

- Launcher icons at multiple densities.
- XML drawables for launch and window backgrounds.
- Android value resources such as colors and styles.

## When To Edit This

- Edit this folder when Android branding, launch screen, or native resource values change.
- Use the parent README instead of adding separate docs to every density bucket.

## Related Folders

- `assets/images/` stores Flutter-rendered image assets.
- `assets/branding/` stores source branding material.
