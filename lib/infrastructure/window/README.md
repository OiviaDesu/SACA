# Window Infrastructure

## Purpose

This folder configures desktop window behavior for SACA.

## What Is Here

- IO and stub window configurators.
- Conditional exports for platforms that support desktop window setup.

## When To Edit This

- Edit this folder when desktop window size, title, or shell setup changes.
- Keep responsive layout rules in `lib/core/layout/`.

## Related Folders

- `windows/` and `macos/` are desktop host projects.
- `lib/presentation/screens/` adapts content to window size.
