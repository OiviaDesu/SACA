# Localization Infrastructure

## Purpose

This folder loads language resources from bundled assets.

## What Is Here

- Asset-backed lexicon repository implementation.
- Runtime bridge between data files and domain vocabulary services.

## When To Edit This

- Edit this folder when lexicon loading or asset format changes.
- Keep visible UI copy in `lib/presentation/localization/`.

## Related Folders

- `assets/data/` stores lexicon data.
- `lib/presentation/localization/` stores UI strings.
