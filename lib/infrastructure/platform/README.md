# Platform Infrastructure

## Purpose

This folder contains platform-specific policy that does not fit a feature adapter.

## What Is Here

- Desktop shell policy and related platform decisions.

## When To Edit This

- Edit this folder when platform shell behavior changes.
- Keep UI components and theme rendering in `lib/presentation/`.

## Related Folders

- `lib/infrastructure/window/` configures desktop windows.
- Platform host folders provide native shells.
