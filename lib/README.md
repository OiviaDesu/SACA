# Flutter Application Code

## Purpose

This folder contains the cross-platform SACA app written in Dart and Flutter.

## What Is Here

- Application use cases that coordinate assessment flow.
- Domain models and clinical policies.
- Infrastructure adapters for ML, speech, localization, web, and desktop.
- Presentation code for screens, controllers, themes, and widgets.

## When To Edit This

- Edit this folder for app behavior, UI, localization, diagnosis flow, or runtime wiring.
- Keep platform-specific shell changes in `android/`, `ios/`, `macos/`, or `windows/`.

## Related Folders

- `assets/` provides runtime images, data, and model bundles.
- `test/` validates this code.
