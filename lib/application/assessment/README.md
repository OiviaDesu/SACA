# Assessment Use Cases

## Purpose

This folder contains the application actions for one SACA assessment session.

## What Is Here

- Start, input, voice transcript, question answer, and analysis use cases.
- Shared result types that describe state transitions for the controller.

## When To Edit This

- Edit this folder when the sequence of assessment actions changes.
- Keep UI rendering outside this folder.

## Related Folders

- `lib/domain/services/` provides policies used by these use cases.
- `lib/presentation/screens/` renders the assessment flow.
