# Layout Policy

## Purpose

This folder defines reusable adaptive layout rules for SACA.

## What Is Here

- Window size classes.
- Policies for adapting UI to phone, tablet, desktop, and web sizes.

## When To Edit This

- Edit this folder when responsive breakpoints or layout policy changes.
- Keep individual widget layout details in `lib/presentation/`.

## Related Folders

- `lib/presentation/screens/` uses these policies to build screens.
- `web/` and desktop platform folders rely on responsive UI behavior.
