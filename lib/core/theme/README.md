# Theme Primitives

## Purpose

This folder defines shared theme data used by SACA UI components.

## What Is Here

- Theme colors, material tokens, and visual style definitions.
- Semantic roles used by Modern, Glass, and Classic renderers.

## When To Edit This

- Edit this folder when theme tokens or shared visual semantics change.
- Keep component rendering details in `lib/presentation/widgets/`.

## Related Folders

- `lib/presentation/widgets/` consumes theme primitives.
- `lib/presentation/settings/` lets users choose theme options.
