# SACA Flow Screen Parts

## Purpose

This folder splits the main assessment screen into smaller readable widget files.

## What Is Here

- Input, question, review, settings, result, and shared layout widgets.
- Desktop shell and adaptive layout helpers for the assessment flow.

## When To Edit This

- Edit this folder when the visible assessment journey changes.
- Keep common buttons, sliders, and controls in `lib/presentation/widgets/`.

## Related Folders

- `lib/presentation/screens/saca_flow_screen.dart` composes these parts.
- `lib/presentation/widgets/` provides shared UI components.
