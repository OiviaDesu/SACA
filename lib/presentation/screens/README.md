# Screens

## Purpose

This folder contains full-screen Flutter UI for the SACA app.

## What Is Here

- Main assessment flow screen.
- Screen-level widgets split into focused files for input, questions, review, settings, and results.
- Layout wrappers used by desktop and mobile presentations.

## When To Edit This

- Edit this folder when screen structure, page flow visuals, or result presentation changes.
- Keep reusable controls in `lib/presentation/widgets/`.

## Related Folders

- `lib/presentation/controllers/` provides screen state.
- `lib/application/assessment/` handles screen actions.
