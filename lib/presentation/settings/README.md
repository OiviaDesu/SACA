# Settings Presentation

## Purpose

This folder contains settings-related presentation helpers.

## What Is Here

- Code that supports language, theme, and app preference UI.

## When To Edit This

- Edit this folder when settings behavior or presentation changes.
- Keep durable settings state and domain rules outside this folder if they grow.

## Related Folders

- `lib/presentation/screens/saca_flow/` renders settings panels.
- `lib/core/theme/` defines theme values.
