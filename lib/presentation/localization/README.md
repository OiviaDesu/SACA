# UI Localization

## Purpose

This folder stores visible UI language for English and Gurindji modes.

## What Is Here

- Localizer API used by widgets.
- English and Gurindji string catalogs for screens, buttons, dialogs, and results.

## When To Edit This

- Edit this folder when user-visible copy changes.
- Keep research source lexicons in `assets/data/` or `python_pipeline/data/raw/`.

## Related Folders

- `test/saca_localizer_test.dart` verifies localization coverage.
- `lib/infrastructure/localization/` loads runtime lexicon assets.
