# Presentation Layer

## Purpose

This folder contains SACA's Flutter user interface.

## What Is Here

- Controllers that bind app state to widgets.
- Localized UI copy.
- Screens, settings, readiness UI, adaptive style, and reusable widgets.

## When To Edit This

- Edit this folder for visible UI, theme behavior, copy, dialogs, and user interactions.
- Keep clinical decisions and model logic outside the presentation layer.

## Related Folders

- `lib/application/` receives UI actions.
- `lib/core/theme/` defines visual tokens consumed by widgets.
