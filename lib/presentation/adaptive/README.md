# Adaptive Presentation

## Purpose

This folder contains UI policy for adapting presentation style by platform.

## What Is Here

- Platform style helpers used by SACA widgets and screens.

## When To Edit This

- Edit this folder when platform-specific visual behavior needs to change.
- Keep layout breakpoints in `lib/core/layout/`.

## Related Folders

- `lib/presentation/widgets/` consumes style decisions.
- `lib/core/theme/` provides theme tokens.
