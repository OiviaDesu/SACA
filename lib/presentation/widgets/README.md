# Shared Widgets

## Purpose

This folder contains reusable Flutter widgets used across SACA screens.

## What Is Here

- Buttons, chips, sliders, dialogs, overlays, panels, and theme-aware controls.
- Shared visual components that respect Modern, Glass, and Classic styles.

## When To Edit This

- Edit this folder when reusable UI behavior or component styling changes.
- Keep screen-specific layout in `lib/presentation/screens/`.

## Related Folders

- `lib/core/theme/` defines semantic visual tokens.
- `test/widget_test.dart` validates key UI behavior.
