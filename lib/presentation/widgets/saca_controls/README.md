# SACA Control Parts

## Purpose

This folder splits reusable SACA controls into focused implementation files.

## What Is Here

- Theme-specific control renderers.
- Button, field, slider, panel, and dialog building blocks.

## When To Edit This

- Edit this folder when shared controls need visual or interaction changes.
- Preserve public widget APIs where possible so screens and tests stay stable.

## Related Folders

- `lib/presentation/widgets/` exports and composes shared controls.
- `lib/core/theme/` supplies control colors and material tokens.
