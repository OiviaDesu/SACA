# Project Tools

## Purpose

This folder contains developer tools and local demo scripts for SACA.

## What Is Here

- Local web demo server utilities.
- Documentation and validation helper scripts.
- One-off tools that support development but are not app runtime code.

## When To Edit This

- Edit this folder when local workflows, demo servers, or repo validation helpers change.
- Keep Flutter runtime logic in `lib/`.

## Related Folders

- `web/` contains static web host files.
- `python_pipeline/` contains research pipeline tooling.
